# Telegram Bot

Telegram Bot is a NodeJS Project for telegram bot server written in js.

## Installation

Use the package manager [node](https://nodejs.org/en/download/package-manager/) to install Telegram Bot.

```bash
npm install
```

## Usage

```python
npm start

#chat with bot created

hi
# returns 'hello'

# and so many replies can be added in the code
```
